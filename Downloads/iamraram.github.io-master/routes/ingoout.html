비공식 선린인터넷고등학교 인트라넷 (개발 중단)
무슨 서비스인가요?
인고아웃은 교내 커뮤니티, 시간표, 급식 등을 제공하는 비공식 교내 인트라넷으로 디미고의 디미고인, 한세사이버보안고의 한빛과 같은 통합 인트라넷 구축에 의미를 두었습니다. 프론트엔드의 90%를 개발한 후 백엔드를 개발하던 중 공식 인트라넷이 이미 개발중이라는 말을 듣고 중단된 프로젝트입니다.
누가 개발했나요?
  • 디자이너: 백시현, 표한빈, 박정우
  • 개발자: 이하람, 주현명, 성예건
어떤 기능이 있고, 어떻게 개발했나요?
  • Svelte를 이용해 프론트엔드를 개발했습니다.
  • Live share, Code with me, Gitflow등을 이용한 협업을 진행했습니다.
  • 나이스 API와 선린인터넷고등학교 홈페이지 Crawling을 통해 급식과 시간표를 불러옵니다.
