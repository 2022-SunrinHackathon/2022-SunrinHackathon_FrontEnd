JEKYLL로 만든 개발자 블로그
무슨 서비스인가요?
DB를 다룰 줄 몰랐던 때, 네이버나 티스토리같은 이미 만들어져있는 블로그를 쓰는 것보다 무엇이라도 직접 만든 블로그를 쓰는 것이 좋다고 생각했습니다. 그러던 중 JEKYLL이라는 정적 사이트 생성기를 알게 되었고, CSS를 일부 수정해서 사이트를 완성했습니다.
어떤 기능이 있고, 어떻게 개발했나요?
  • JEKYLL을 이용해 정적 사이트를 만들었습니다.
  • Github pages로 정적 페이지를 로드했습니다.
