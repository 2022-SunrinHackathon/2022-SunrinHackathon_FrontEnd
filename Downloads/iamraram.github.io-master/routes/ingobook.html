INGOBOOK
교내 교과서 PDF 다운로드 서비스 (개발 중)
무슨 서비스인가요?
선린인고에서 기존에 PDF를 다운로드 받으려면 각 선생님을 찾아가 파일을 다운받아야 했습니다. 이 점을 해결하고자 모든 교과서 PDF를 모아 다운로드하는 서비스를 개발 중입니다. 궁극적으로는 학습지를 제출하고 선린인고를 종이를 쓰지 않는 학교로 만드는 것이 목표입니다.
누가 개발했나요?
  • 디자이너: 백시현, 박시원, 박정우, 표한빈, 오지후, 문찬솔
  • 개발자: 이하람, 이민규, 주현명, 윤도현,
어떤 기능이 있고, 어떻게 개발했나요?
  • Svelte를 이용해 프론트엔드를 개발했습니다.
  • Mysql을 이용해 데이터를 관리합니다.
  • Postman, Replit, Notion등을 사용해 협업의 효율성을 높이고 있습니다.
  • Salt, Token, Hash 등으로 유저 정보를 안전하게 저장하는 법을 고민 중입니다.
